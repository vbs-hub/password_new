🛠 Команды меню
выход — Безопасное завершение программы.

создать — Запуск мастера генерации (Случайный или Умный пароль).

сохранить — Запись собственного пароля в базу.

открыть — Быстрое открытие файла password.txt через стандартный блокнот.

история — Просмотр 3 последних созданных паролей.

поиск — Поиск пароля в базе по его уникальному ID.

📂 Структура проекта
g_p_v3-alpha.py — Исходный код программы.

g_p_v3-alpha.exe — Скомпилированная версия для Windows.

password.txt — (Создается автоматически) Ваша локальная база паролей.

⚠️ Безопасность
Данная программа хранит пароли в открытом текстовом виде (.txt). Используйте её для обычных сайтов. Для критически важных данных (банки, основные почты) рекомендуется использовать менеджеры с шифрованием.
